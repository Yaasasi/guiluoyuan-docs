# 洛书南社区服务条款


## 概述

洛书南社区（Losenone Community），是一个由众多玩家构成的 Minecraft 中文社区聚落。我们一直秉承着高素质、高质量和正能量的发展理念，致力于打造一个良性、优质与美好的 Minecraft 社区！

归落原（“Ourmoe”、“我们” 或 “我们的”）致力于融合所有热爱生活、正直向上与友好善良的人群，努力与所有社区成员们一起构建欢乐且让人留恋的兴趣社群，构建温暖、和谐、友爱、美好、多元的社区，打造主题鲜明、内容真实的社区，鼓励有爱、有趣、有用的内容在社区生长。感谢您参与其中，与我们共同建设！

如果您对《归落原服务条款 / Minecraft 》（以下简称 “本条款” ）现有的条款有任何的疑问（包括实施中的困难）和建议，请通过邮件 help@service.our.moe 与我们取得联系。

## A类 - 条款的适用、生效与变更

1. 本条款适用于归落原的各类 Minecraft 社区及其所属社区成员和其提供的各类服务、归落原开设的社区服务器及其玩家以及由社区衍生的交流内容、交流行为以及游玩行为（以下简称 “社区约束范围” ），特殊情况将另行注明：\
1）除非特别注明，本条款同样适用于洛书南社区约束范围。但是，若本条款与《[洛书南社区管理条例](losenone_community_management_regulations)》有冲突，请以后者为准。

2. 本条款无论因何种原因部分无效，其余条款仍有效，对各方具有约束力。

3. 本条款的所有标题仅为阅读方便，本身并无实际涵义，不能作为本条款涵义解释的依据。

4. 如若发生了因语言翻译产生的条款理解问题，请您以本条款的简体中文版为准确版本。

5. 如果您目前在中国大陆地区以外，您接受与履行本条款还需要同时遵守您所属和（或）所处国家或地区的法律。

6. 如果您在本条款适用的范围中违反了相关法律法规或本条款，相关国家机关或机构可能会对您提起诉讼、罚款或采取其他制裁措施，并要求归落原给予协助。因此给您或者他人造成损害的，您应自行承担全部责任，归落原不承担任何责任。您因违反本条款导致任何主体产生损失，您应当独立承担责任；归落原因此遭受损失，您也应当一并赔偿。

7. 请您确保您在加入归落原的各类 Minecraft 社区（以下简称 “社区” ）前与使用社区提供的服务前已仔细阅读、充分理解并毫无保留地接受本条款所有条款，否则您无权加入社区以及使用社区提供的服务。您加入社区即视为您已阅读本条款并接受本条款的约束。您点击同意、接受或下一步，或您注册、使用社区相关服务均视为您已阅读并接受本条款的约束。

8. 您在使用社区提供的服务时应同样遵守社区发布的其他规则。您接受并同意本条款的行为即视为您接受并同意社区发布的其他规则，您接受并同意任一社区发布的规则的行为亦视为您接受并同意本条款。

9. 归落原有权在必要时修改本条款的内容，您可以在「[归落原 · 文档](../contents)」中查阅最新版本的条款。本条款变更后，如果您仍在使用社区提供的服务，即视为您已接受变更后的条款。

10. 本条款已于 2024 年 03 月 15 日更新。请花一些时间熟悉我们的隐私政策，如您有任何问题，请联系我们。


## B类 - 社交环境

在本类中，我们实行金字塔阶级处罚措施（共五级），高阶级包含次一级的处罚措施，以此类推：\
   • 第一级：法律手段；\
   • 第二级：永久封禁（社区公开）；\
   • 第三级：限时禁言（24 小时、7 个自然日、1 个自然月）；\
   • 第四级：严重警告（社区通报）；\
   • 第五级：消除影响（撤回或删除）。\
单一社区成员每遭到一次处罚，就会提高一次处罚等级。但是，处罚等级的提高不是绝对线性提高的，我们会根据实际情况的严重程度动态变更，同时我们也会注明其是否以何种形式提高（“0” 代表此前并无处罚等级，即初次受到处罚）：\
   • 线性提高（按照处罚阶级次序提高处罚等级）；\
   • 动态提高（按照实际情况动态提高处罚等级）；\
   • 固定阶级（固定的处罚阶级）。

1. 您在社区约束范围内参与社交活动时必须遵守《[公共社区交流公约](../treaty/community_communication_covenant)》、中华人民共和国的法律法规以及您所处国家的法律法规。

2. 高质量昵称与头像，传递你有趣的灵魂。符合社区及个人特色的昵称和头像可以帮助大家展示自己的风采，快速融入新群体。您对外展示的个人名片不得违反[《公共社区交流公约》第四项 1、2、3、4 款之规定](https://www.our.moe/docs/treaty/community_communication_covenant#%E7%AC%AC%E5%9B%9B%E9%A1%B9-%E4%B8%AA%E4%BA%BA%E5%90%8D%E7%89%87)。\
   • 处罚等级提高方式：线性提高（0 -> 5 -> 4 -> 3 -> 2 -> 1）；\
   • 处罚执行管理员：社区秩序管理员。

3. 在社区约束范围内，不得违反[《公共社区交流公约》第五项 1、2、3、4、5、6 款之规定](https://www.our.moe/docs/treaty/community_communication_covenant#%E7%AC%AC%E4%BA%94%E9%A1%B9-%E4%BA%A4%E6%B5%81%E8%A1%8C%E4%B8%BA)。\
   • 处罚等级提高方式：动态提高（0 -> 3 -> 2 -> 1）；\
   • 处罚执行管理员：社区秩序管理员。

4. 在社区约束范围内，不得违反[《公共社区交流公约》第五项 7、8、9、13、14、15、16、17、18、19、20、21 款之规定](https://www.our.moe/docs/treaty/community_communication_covenant#%E7%AC%AC%E4%BA%94%E9%A1%B9-%E4%BA%A4%E6%B5%81%E8%A1%8C%E4%B8%BA)。\
   • 处罚等级提高方式：线性提高（0 -> 5 -> 4 -> 3 -> 2 -> 1）；\
   • 处罚执行管理员：社区秩序管理员。

5. 在社区约束范围内，不得违反[《公共社区交流公约》第五项 10、11、12 款之规定](https://www.our.moe/docs/treaty/community_communication_covenant#%E7%AC%AC%E4%BA%94%E9%A1%B9-%E4%BA%A4%E6%B5%81%E8%A1%8C%E4%B8%BA)。\
   • 处罚等级提高方式：动态提高（0 -> 5 -> 4 -> 3 -> 2）；\
   • 处罚执行管理员：社区秩序管理员；\
   • 处罚豁免注明：相关社区管理团队授权的行为，可豁免此项处罚。

6. 在社区约束范围内，不得引导其他社区成员加入任何与本社区无关的社群，同时您也不应该在与本社区相同领域的社群宣传本社区。\
   • 处罚等级提高方式：动态提高（0 -> 5 -> 3 -> 2）；\
   • 处罚执行管理员：社区秩序管理员。

   • 线性提高（按照处罚阶级次序提高处罚等级）；\
