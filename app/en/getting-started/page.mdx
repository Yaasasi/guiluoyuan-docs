# Getting Started

## Local Development

If this is your first time running the project, install dependencies first:

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`, it will redirect to `http://localhost:3000/zh` by default.

## Production Build

If this is your first time running the project, install dependencies first:

```bash
pnpm install
pnpm build
pnpm start
```

After `build`, the `postbuild` script generates a full-text index in `public/_pagefind`.
